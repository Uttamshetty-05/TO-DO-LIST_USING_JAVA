# 🧾 CLI To-Do List App (Java)

A simple command-line To-Do List application built using Java.

## 🚀 Features
- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks

## 🛠 Tech Used
- Java
- ArrayList
- OOP Concepts

## ▶️ How to Run

1. Compile:
   javac TodoApp.java

2. Run:
   java TodoApp
